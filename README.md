# 💰 PrestControl

Aplicación web PWA para gestión de préstamos personales. Funciona directamente en el navegador sin necesidad de instalación.

## Características

- Registro y seguimiento de préstamos
- Control de cuotas y pagos
- Estadísticas y gráficos
- Exportación a PDF y Excel
- Modo offline (PWA)
- Sincronización con Firebase / Firestore
- Diseño responsive para móvil y escritorio

## Tecnologías

- HTML5 / CSS3 / JavaScript (Vanilla)
- [Firebase](https://firebase.google.com/) — Auth + Firestore
- [Chart.js](https://www.chartjs.org/) — gráficos
- [jsPDF](https://github.com/parallax/jsPDF) — exportación PDF
- [SheetJS](https://sheetjs.com/) — exportación Excel

## Uso

Abrí `index.html` directamente en el navegador, o desplegá como [GitHub Pages](https://pages.github.com/).

## Despliegue en GitHub Pages

1. Subí el repositorio a GitHub
2. Andá a **Settings → Pages**
3. En *Source* elegí la rama `main` y carpeta `/ (root)`
4. GitHub te dará la URL pública en minutos
